*Sitio web de 5 (cinco) pages: index - sobremi - recetas - articulos - contactame.

*La estructura general del sitio se encuentra dividida en tres grandes bloques, header/main/footer. El header y el footer cuentan de contenido fijo, mientras que el contenido
main varia de acuerdo a la pagina, exceptuando por el aside incorporado dentro del mismo, que tambien es fijo.

*Todo el sitio se encuentra maquetado con Bootstrap en cuanto a la estructura del mismo.

*Los estilos del sitio, se han realizado a traves Sass, dividiendo las hojas de estilos en partials correspondiente a cada bloque de contenido, variables y generales. Todos 
los anteriores se encuentran canalizados a traves del archivo Sass "styles" importando cada uno de los partials.

*Las fuentes son incorporadas mediantes import (web) mientras que las img se encuentran trabajadas de manera local. 

*Las animaciones se encuentran alojadas en el partial "general" mientras que los comandos de ejecucion en la seccion correspondiente. 

*El sitio se encuentra versionado a traves de Git/Github.

*Servidor utilizado: Github Pages, en linea a traves de la rama master "main". Las versiones se utilizaron para trabajar el sitio.
